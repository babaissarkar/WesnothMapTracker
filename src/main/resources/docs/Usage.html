• Click Open Background from the File menu to select a background to draw the markers on.

  • Click Add more markers button and select Battle for Wesnoth misc directory (the one with the new journey/new rest markers etc).
    Supported Markers
    new-journey.png=NEW_JOURNEY
    dot-white.png=OLD_JOURNEY
    new-battle.png=NEW_BATTLE
    new-battle2.png=NEW_BATTLE
    cross-white.png=OLD_BATTLE
    flag-red.png=NEW_REST
    flag-red2.png=NEW_REST
    flag-white.png=OLD_REST
  • Save using File > Save once you are done. The output is like this:
    {NEW_JOURNEY 455 300}
    {NEW_REST 500 300}
